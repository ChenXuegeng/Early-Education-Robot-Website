# 幼教启蒙智能家务机器人社区 (Smart Robot Community)

这是一个为幼教启蒙智能家务机器人设计的社区网页原型。

## 功能特点

- **社区首页**: 查看大家分享的机器人趣事和编程挑战。
- **发布帖子**: 分享你的创意和成果。
- **积分中心**: 查看积分（官方激励 & 家庭激励）并兑换奖励。
- **我的资料**: 管理个人信息和查看历史动态。
- **积分机制**: 当你的帖子被别人点赞时，你将获得积分。
 - **挑战与视频评论**: 官方任务页包含三项挑战（最优路径规划、物流搬运、避障）。每个挑战下可发布评论并上传视频，评论被点赞即可集赞得积分。

## 运行项目

1.  安装依赖:
    ```bash
    npm install
    ```

2.  启动开发服务器:
    ```bash
    npm run dev
    ```
    11

3.  打开浏览器访问显示的地址 (https://chenxuegeng.github.io/Early-Education-Robot-Website/)。

## 模拟测试

为了演示“别人点赞我获得积分”的功能，在**官方任务**页每条由你发布的评论下都有 **“模拟别人点赞我 +10积分”** 按钮。点击它可模拟他人点赞你的视频评论，并为你增加积分。
